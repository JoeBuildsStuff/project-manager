# flags

The bitwise `OR` of zero or more key import/export flags.

macOS 10.0+

```
var flags: SecKeyImportExportFlags
```

## [Discussion](https://developer.apple.com/documentation/security/secitemimportexportkeyparameters/flags\#Discussion)

Use the values from [`SecKeyImportExportFlags`](https://developer.apple.com/documentation/security/seckeyimportexportflags).
